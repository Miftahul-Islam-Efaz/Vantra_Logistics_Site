# Vantra Logistics

### Freight that doesn't wait.

A concept freight and container logistics brand — designed to explore what a
premium, editorial-grade logistics experience could look and feel like on the web.

## Overview

Vantra Logistics is a front-end design and engineering showcase — not an
operating logistics company. Every section was built to make a traditionally
utilitarian industry feel precise, modern, and cinematic: motion-driven hero
sequences, container spec breakdowns, real-time-feeling stats, and a global
hub network visualization.

## Key Sections

**Hero — "Freight that doesn't wait."**
Full-bleed motion sequence, animated truck reveal, live-feeling shipment and
coverage stats.

**Container Catalog**
Dimension breakdowns for Standard 20 DC, Standard 40 DC, and High Cube 40 HC,
plus condition tiers (New / Cargo Worthy).

**"Every mile, driven by care."**
An editorial break in the data-heavy flow — driver-first messaging backed by
fleet stats (drivers, on-time rate, years of experience, missed handoffs).

**Global Hub Network**
Footer-level visualization of terminal locations across North America, the
Pacific, and Europe.

<br>

## Tech Stack
Frontend        React
Animation       GSAP / motion-driven transitions
Rendering       Prerendered SPA (react-snap) for crawler/SEO accessibility
Styling         Custom design system — dark base, orange accent, editorial type

<br>

## Design Language

Pure black backgrounds, high-contrast condensed display type, a single
disciplined orange accent, and cinematic photography-led sections — built to
read as premium rather than generic-corporate.
